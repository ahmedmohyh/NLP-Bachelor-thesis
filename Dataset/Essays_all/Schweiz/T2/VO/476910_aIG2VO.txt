Despite the fact that computers have seemingly started to take over the world, one might think that by now everything must be electronic. What concerns our voting system, that would not be the case. After many elections, human error has been gradually decreased to a (debatable) sustainable amount, which means that humans got better at collecting voting data safely.  Sometimes mistakes still happen, but the consquences of an error in the current voting system would be smaller than if there was an error in the code. Arguing that the code had to be written by a human, there could always be an error in the code somewhere and having an entire election rely on the on such a probability seems unstable. However, computers are trusted with some of the most valuable information on earth and many methods have been employed to ensure data security.
